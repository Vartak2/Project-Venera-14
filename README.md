#  Введение в авиационную и ракетно-космическую технику

### Команда: ***Орбита***
### Миссия: ***Венера-14***

# Описание
= Код из файла `KSP_speed.py` строит график отношения *скорости* от времени во время выхода ракеты на околоземную орбиту и параллельно собирает данные в csv-файл.
= Код из файла `KSP_hight.py` строит график отношения *высоты* от времени во время выхода ракеты на околоземную орбиту и параллельно собирает данные в csv-файл.
